How to upload a file to an Azure storage account

15 October 2014
  1. Create a storage account on the Azure website and create a container (like a folder).
  2. In Azure, go to your storage account, the configure tab, and click 'manage access keys' at the bottom.
  3. Download Cyberduck. Install it. Run it
  4. Open a new connection and enter the details from the Azure screen you have open.
